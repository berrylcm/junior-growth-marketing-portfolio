# Junior Growth Marketing Specialist Portfolio Setup

## Overview
This repository documents my process of setting up an AI-assisted
development environment using Cursor IDE, Claude Code, and Codex
as part of the application process for the Junior Growth Marketing
Specialist role at 100Hires.

## Tools Installed
- **Cursor IDE**, AI-native code editor
- **Claude Code** (VS Code extension), Anthropic's AI coding assistant
- **Codex**, OpenAI's coding agent

## Steps Completed
1. Downloaded and installed Cursor IDE
2. Signed up and verified email for Cursor account
3. Opened Cursor from desktop
4. Installed Claude Code extension via Extensions panel
5. Installed Codex extension via Extensions panel
6. Created a GitHub account and set up a public repository
7. Cloned the repository into Cursor using `git clone`
8. Documented the process and used Claude to format this README

## Challenges & How I Solved Them
Coming from a non-IT background, some steps required extra research:
- Unfamiliar with Git clone syntax and overall workflow. I asked Claude directly to understand better tools and workflow
- Could not verify whether Claude Code was the active model in Cursor, as model selection requires a Pro subscription. But I proceeded based on the fact that the extension was installed successully
